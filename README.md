## 1 - Requisitos

* Node 17 ou superior

## 2 - Compilação

<p>Instalação dos modulos:</p>

```sh
npm install
```

<p>Para executar o servidor em conjunto com o compilador TS em watch:</p>

```sh
npm run start
```
