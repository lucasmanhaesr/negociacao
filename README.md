## 1 - Requisitos

* Node 17 ou superior

## 2 - Compilação

<p>Para compilar TS para JS:</p>

```sh
npm run compile
```

<p>Para executar o servidor:</p>

```sh
npm run server
```

<p>Para executar o servidor em conjunto com o compilador TS em watch:</p>

```sh
npm run start
```
